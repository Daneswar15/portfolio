## 🚀 Getting Started

### Prerequisites
- Git (for version control)
- A web browser (to view the site)

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/Daneswar15/portfolio.git
cd portfolio
```

2. Open the portfolio locally:
- Simply double-click `index.html` or
- Run a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if you have http-server installed)
http-server
```

3. Visit `http://localhost:8000` in your browser

## 📦 Deployment

### GitHub Pages (Recommended)

1. Your repository is already set up for GitHub Pages
2. Push your changes to the `main` branch:
```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

3. Enable GitHub Pages:
   - Go to your repository settings
   - Navigate to "Pages" section
   - Select "main" branch as source
   - Wait for the site to deploy (usually takes a few minutes)

4. Access your portfolio at: `https://Daneswar15.github.io/portfolio`

### Other Deployment Options

**Netlify:**
- Connect your GitHub repository to Netlify
- Netlify automatically deploys on every push

**Vercel:**
- Similar to Netlify, connect your repository
- Automatic deployments with preview URLs

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #00d4ff;
    --secondary-color: #0084d4;
    --dark-bg: #0a0e27;
    --darker-bg: #050810;
    /* ... other colors */
}
```
